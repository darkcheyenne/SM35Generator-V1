SM35 - Berechtigungsrollen umbenennen

Kopieren Sie mit CTRL+C und CTRL+V die Liste der Rollen, welche Sie umbenennen möchten in das Feld unterhalb.
Tipp: Wenn Sie die Daten in ein Excel-Dokument schreiben (Spalte A = Alte Rolle, Spalte B = Neue Rolle), dann beide Spalten markieren und dann in das Textfeld unterhalb kopieren, dann haben Sie automatisch die richtige Schreibweise.
